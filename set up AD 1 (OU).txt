2. create organizational units (OUs) for regions
	OPTION1 (GUI):
	in active dir. go to your server domain
	right-click on your server domain
	click new
	click organizational unit
	add the name
	click ok
	// you can add OU inside OU
	
2. create groups in (OUs)
	OPTION1 (GUI):
	right-click on your OU
	click new
	click group
	add the name
	// in group scope we have options for group (uni. , global, domain local) (usually we use global)
	// in group types we have 2 options for group (security or distribution) (usually we use security)
	click ok
	
2. create users in (OUs)
	OPTION1 (GUI):
	right-click on your OU
	click new
	click user
	add the Firstname, Lastname 
	add User loggon name
	click next
	add password
	// you can (✓) any of the check boxes
	click finish
	
//you can move any computer in AD to the OU you want by:
	right click on the computer
	click move
	choose the OU you want then click ok
